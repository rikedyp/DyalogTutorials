    Development Environment

    Dyalog provides two integrated development environments (IDEs). There is the Dyalog IDE for Microsoft Windows, and the Remote IDE (RIDE). The RIDE can be used on all platforms, but is the only GUI available for Linux and macOS. As the name suggests, the RIDE also provides remote interaction and debugging capabilities over a network.
